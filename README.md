# plum

**Rhode Island's plumber, on demand.**

AI-powered residential plumbing platform serving Woonsocket, Providence, Smithfield & N. Smithfield, RI.

---

## About

plum is a full-stack plumbing service platform with an AI chat assistant that helps customers book visits, choose protection plans, and get emergency support — all through a conversational interface powered by Gemini function calling.

## Features

- **AI Chat Assistant** — Gemini-powered conversational booking with autonomous function calling
- **Smart Scheduling** — Technician matching based on availability, specialty, and location
- **Protection Plans** — Subscription tiers (Basic $29/mo, Standard $49/mo, Premium $79/mo)
- **Technician Dashboard** — Admin and tech views for managing appointments
- **Emergency Dispatch** — Priority routing for urgent plumbing issues

## Tech Stack

- **Frontend** — Next.js 15, React 19, Tailwind CSS 4, Framer Motion
- **Backend** — Next.js API Routes, Gemini (function calling)
- **Database** — Supabase (PostgreSQL)

## Getting Started

```bash
npm install
cp .env.local.example .env.local  # add your API keys
npm run dev
```
